# Promo Rules Guide

Статический сайт-гайд для подготовки правил маркетинговых акций.

## Что внутри

- инструкция для создания Jira-задачи по шаблону `[P&C] Communications`;
- промпт для ChatGPT;
- полные примеры правил, зашитые в сайт и промпт.

## Как открыть локально

Откройте `index.html` в браузере.

## Как опубликовать

Проект можно опубликовать как статический сайт через GitHub Pages, Vercel, Netlify или любой простой хостинг.
