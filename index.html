Инструкция

Порядок работы

  1. Заполните задачу на доске запусков акций. Создайте задачу с типом Task, в поле Template выберите [P&C] Communications. После выбора шаблона в описании задачи появятся поля, которые нужно заполнить: даты, ЦА, механика, промокод, призовой фонд, лимиты, исключения и сроки начисления. Открыть доску
  2. Скопируйте промпт и вставьте его в ChatGPT. После ввода промпта ChatGPT должен ответить, что готов получить данные о новой акции из задачи.
  3. Вставьте текст из задачи. Если данных не хватает, GPT сначала задаст уточняющие вопросы.
  4. Проверьте черновик перед публикацией. Сверьте даты, промокод, событие, призовой фонд, лимиты, исключения и юридический блок.
